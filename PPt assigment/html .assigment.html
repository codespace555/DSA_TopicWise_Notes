Q.1 "< DOCTYPE html >" is it a tag of html? If not, what is it and why do we use it?

Answer :-"< DOCTYPE html > "is not a tag in HTML; it is called a document type.The doctype declaration for HTML5, which is the current version of HTML. The < DOCTYPE html > must be placed at the very beginning of an HTML document, before the < html > tag. This declaration tells the browser that the document is written in HTML5 syntax By providing this information, the browser can switch to standards mode.

Q.2 Explain Semantic tags in html? And why do we need it?

Answer :- Semantic tags in HTML are describe the purpose or role of the content they contain, making it easier for search engines, assistive technologies, and developers to understand the page's structure and improve accessibility.for Example :-<header>,<nav>,<section>,<footer>,<main>.They help screen readers and other devices to navigate and understand the content more accurately.This can potentially improve SEO rankings and visibility.Semantic tags make the HTML code more readable and maintainable for developers

Q.3 Differentiate between HTML Tags and Elements?

Answer : -
Differentiate between HTML Tags and Elements
HTML Tags HTML Elements
HTML tags are like markers that indicate the beginning and end of an HTML element. They are written using angle bracket,< tags > . Examples:- <p>,< div >,< h1 > . HTML elements are the complete entities that include the opening tag, content, and closing tag. They represent specific parts of the content or structure within an HTML document. For example, <p>Some text </p> is an HTML element where <p> "Some text" </p> is the closing tag.
